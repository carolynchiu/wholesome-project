# 使用技術
  * 設計/原形工具：Figma
  * 前端：HTML, CSS, Javascript, Axios, Bootstrap 5, SASS
  * 前端框架：React
  * 後端技術：Node.js 16.16.0、Express
  * 資料庫運用/版本控制：MySQL、Git

###### 使用套件：
  * framer-motion@7.3.6
  * mui material @5.10.3
  * swiper @8.3.2
  * sweetalert2 @11.4.33
